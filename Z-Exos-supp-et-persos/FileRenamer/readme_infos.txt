Lisez moi avant d'executer le programme.

FileRenamer permet de renommer des fichiers récurrents (de notes par exemple) avec une structure particulière. La structure se choisit dans le formulaire, puis se sauve dans une préconfiguration en local.

Prérequis:
Le répertoire donné est le lieu ou les fichiers seront traités. Attention les sous dossiers sont aussi traités.....a corriger.


Les fichiers sont nommés de manière simple dans le dossier du cours.
